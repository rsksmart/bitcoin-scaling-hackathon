Payment Remittance

Decentralised Payment Remittance platform powered by Rootstock.

Features

SENDER

  • able to lock funds for remittance
  • able to view funds that are remitted but not withdrawn
  • able to claim back funds if receiver does not withdraw within expiry date

RECEIVER

  • able to view funds that are pending withdrawal
  • able to withdraw funds that were remitted to user
